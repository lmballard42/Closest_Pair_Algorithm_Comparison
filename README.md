## Problem Description

This project explores and compares the different run times of different algorithms solving the closest pair problem. This is a problem that finds the two closest points in a metric space, typically using Euclidean distance, for a set of n points.
## Implemented Algorithms

- Brute Force
- Divide and Conquer
- Line Sweep
- Bee Colony Optimization (BCO)
 
## Datasets

Each of our datasets are a group of randomly generated X,Y pairs with a size of n. n is increased exponentially from 2 up to 8191, and there are 5 datasets for each size. They can be viewed in the 'Code' folder under 'Datasets'.
## Run

To run from the command line, navigate to the folder where 'main.py' is located and type

```bash
  python main.py
```

This script runs the 4 algorithms multiple times and writes the output to

```bash
  closest_pair_results.csv
```

in the "Results" folder
